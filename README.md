# Personal CV Website - Web Programming Assignment

## Project Information
This project is a personal CV website built as part of the Web Programming course assignment.  
It showcases my profile, education, skills, projects, and contact information in a clean, responsive layout.  
The website also includes a light/dark mode toggle for better user experience.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- Google Fonts (Inter)
- Semantic HTML structure
